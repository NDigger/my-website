Games Development

hmm...

I began making games because I could see how other games work and expected that making games will be quite an easy task which doesn't require much skill. I was wrong and right at once.

To make games I only tried to use Unity and Godot. I chose Godot because I didn't know how does C# work and I just wanted everything to be as simple as possible.

Making games is not only about programming. You also need reasonable music, textures, and time to combine all this and make it look good. Scripts are more about to program everything you need, create scenes where you can put all the content. Then it's just about "Coding" simple mechanics like "crafting various items" or "what will happen if you talk with specifically this character".

I don't have much experience for now in here. And I'm not sure I will obtain more of it in future, because I'm not good at drawing to create a lot of textures which will fit each other and create the entire working game.

Anyway I created few playable projects so I tried myself in it.
